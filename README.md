# DHCP Configuration in Cisco Packet Tracer

## Overview
This project demonstrates the configuration of DHCP (Dynamic Host Configuration Protocol) in Cisco Packet Tracer. DHCP automatically assigns IP addresses to connected devices.

## Objectives
- Understand DHCP functionality
- Configure automatic IP assignment
- Reduce manual IP configuration
- Test successful communication

## Tools Used
- Cisco Packet Tracer

## Devices Used
- PCs
- Router
- Switch

## Features
- Automatic IP allocation
- Dynamic IP configuration
- Improved network management
- Connectivity verification

## Learning Outcomes
After completing this project, users will understand:
- DHCP configuration process
- Automatic IP assignment
- Client-server communication

## Conclusion
This project provides practical knowledge of DHCP configuration and demonstrates how automatic IP management simplifies networking tasks.
